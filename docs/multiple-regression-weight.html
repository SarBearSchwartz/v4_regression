3.3 REGRESSION ANALYSIS

3.3.1 Fit the models

The dependent variable (DV) is weight loss The independent variables (IVs) are exercise and diet

fit_lm_exer <- lm(loss ~ exer,
                  data = df_loss)

fit_lm_diet <- lm(loss ~ diet,
                  data = df_loss)

fit_lm_both <- lm(loss ~ exer + diet,
                  data = df_loss)
